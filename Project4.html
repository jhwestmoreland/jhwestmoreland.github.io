This project was to strenghten my software skills and combine two of my favorite interests of mine, art and technology.

The Grinch with a Heart 2019

This project was for Christmas 2019.

The nutcracker capabilities are quite simple. Essentially, the user can control an exposed LED, inside the nutcracker.

Overview

I intially was going to make this hardware focused with an external control panel. Instead, I wanted to become more knowledgable about front-end and back-end web design. For this project, the cross platform I used was Apach2 with a python mod. Apache2 is a resource that allows the raspberry pi to act as a web-server. This allows the user to access it on the LAN, instead of having LocalHost:xxxx which is only accessible on the raspberry pi. I have learned a lot testing other cross platforms and just trying to figure out what the best methods.

The Hardware I used was a Raspberry Pi zero, 2 buttons for resetting the apache2 server and powering it off safely, and lastly a LED.
The Software Languages I used were Python, Javascript, HTML5, CSS3, PHP, Bash.
